## 📖 О проекте

Данный проект реализует генерацию пространственно-временных диаграмм для всех 255 правил одномерных клеточных автоматов Вольфрама. Программа позволяет визуализировать эволюцию клеточного автомата во времени, анализировать симметрию полученной диаграммы и сохранять результаты для дальнейшего изучения.

Клеточные автоматы — это дискретные динамические системы, состоящие из регулярной решётки ячеек, каждая из которых может находиться в одном из конечного множества состояний. Эволюция системы происходит дискретно во времени: состояние каждой ячейки на следующем шаге определяется состоянием её соседей на предыдущем шаге в соответствии с локальным правилом перехода.

Стивен Вольфрам систематически исследовал все 255 правил одномерных клеточных автоматов и разделил их на четыре класса поведения:
- **Класс I** — быстрое затухание к однородному состоянию
- **Класс II** — периодические структуры
- **Класс III** — хаотическое поведение
- **Класс IV** — сложное поведение (на границе хаоса и порядка)

## ✨ Возможности

- ✅ Поддержка всех 255 правил Вольфрама (0–255)
- ✅ Два режима начального состояния: `"center"` (центральная ячейка) и `"random"` (случайное распределение)
- ✅ Автоматический анализ вертикальной симметрии диаграммы
- ✅ Сохранение диаграмм в PNG-формат с масштабированием
- ✅ Сохранение статистики экспериментов в CSV-формат
- ✅ Модульное и интеграционное тестирование
- ✅ Контейнеризация через Docker
- ✅ Полная документация в README.md
